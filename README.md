# check_tfoms_errors
Проверка ошибок в реестрах ТФОМС АК  
[CHANGELOG](CHANGELOG.md)

## Windows сборка
Запустить `build_win.bat`.  
Требуются предварительно установленные в ОС:
- Python 3.7+
- PyInstaller (`pip install PyInstaller==3.6`) - для сборки `.exe`
- 7-Zip - для упаковки дистрибутива  

Результат сборки - архив c именем вида `check_tfoms_errors_v0.4.0_dist.zip`

## Использование
1. В директорию `in` поместить 2 архива:  
`HMxxxxxxxxxxx.zip` (или CM*, TM*, DPM*, DVM*, DOM*, DSM*, DUM*, DFM*) (файлы
реестра для загрузки на сервер объединения)  
`Vxxxxxxxxxxxx.ZIP` (файл ответа из ТФОМСа)
2. Запустить исполняемый файл `check_tfoms_errors.exe`  
3. Запустится процесс обработки файлов.  
4. Результат в директории `out`, с настоящим датой и временем.  
5. Также результат будет открыт программой для открытия `CSV` в системе (обычно `MS Excel`).
